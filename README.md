# Draskint

Landing page para empresa de desarrollo web.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Estándar de Commits

Este proyecto usa **Husky** y **CommitLint**.

### Formato

```
<tipo>: <descripción>
```

### Tipos

- `feat` - Nueva funcionalidad
- `fix` - Corrección de bug
- `docs` - Documentación
- `style` - Estilo
- `refactor` - Refactorización
- `perf` - Rendimiento
- `test` - Tests
- `chore` - Mantenimiento

### Ejemplos

```bash
git commit -m "feat: agregar animación de entrada"
git commit -m "fix: corregir validación de email"
git commit -m "docs: actualizar instrucciones"
```
